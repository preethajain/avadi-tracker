Follow these 5 steps to activate live sync for all 3 users. Takes about 5 minutes.
Step 1 — Go to console.firebase.google.com and sign in with a Google account
Step 2 — Click Add project → Name it avadi-tracker → Skip Google Analytics → Create project
Step 3 — In your project dashboard, click the </> Web icon → Register app as avadi-web → Copy the firebaseConfig object shown
Step 4 — In left sidebar click Build → Realtime Database → Create Database → Choose Singapore region → Start in test mode → Enable
Step 5 — Open this HTML file in Notepad/VS Code → Find the section labelled PASTE YOUR FIREBASE CONFIG HERE → Replace each placeholder value with your actual config values → Save → Re-upload to Netlify
